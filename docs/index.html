How to submit homework/projects to Github

  • Visit the Starter files link on the homework card.
  • Click the fork button in the upper right hand corner.
  • The site will now give you an option of where to fork the starter files. Select your name, NOT THE CCS GROUP!
  • In the GitHub desktop application go to File -> Clone Repository
  • You will get a list of all the repos currently in your account. You can even search, pick the one UNDER YOUR NAME
  • Notice you can save the folder anywhere on your system you like. The default is `documents/GitHub` if you can't find it.
  • Congratz! You're now ready to do amazing homewok! Remember to commit AND push back to github
