How to submit homework/projects to Github
using the command line

You can also use the desktop github app but that is your decision and not supported by this guide.

  • For each new project create a new folder exclusively for that project
  • All files for that project should be in the new folder
  • Visit your github account while logged in
  • Select "new repository" and create a new public repo under your name. NOT CCSProgramming
  • And follow the directions on the next page for creating a new repo from the command line
